# Test Data Related to Minecraft 1.18.2+ Nether Fortress Nether Brick Pack Spawning Wander Issue

This is test data left behind while researching the Minecraft 1.18.2+ Nether Fortress Nether Brick Pack Spawning Wander Issue. For the full problem analysis, see Fallen_Breath's article:

https://www.bilibili.com/opus/928484419469049864

https://blog.fallenbreath.me/2024/fortress-nether-bricks-pack-spawning-issue-1182/

## File Contents

`img/` contains some screenshots preserved at the time.

`*.csv` contains test data for various situations. `normal` refers to spawn platforms with wander area, `small` refers to spawn platforms without wander area, `tiny` refers to spawn platforms using carpets placed on part of the fortress interior structure as wander area.
